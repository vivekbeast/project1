AND Gate : A HIGH output results only if all the inputs to the AND gate are HIGH. If not all inputs to the AND gate are HIGH, LOW output results.


OR Gate : The OR gate returns true if either or both of its inputs are true; otherwise it returns false. The input and output states are normally represented by different voltage levels.

NOT GATE : NOT gate is also called an Inverter. The output of NOT gate is always the complement of its input

UNIVERSAL GATES:The NAND and NOR gates are called as universal building blocks. These gates can perform all the three basic logic functions (AND, OR, NOT).
NAND GATE : The NAND gate operates like an AND gate followed by an INVERTER

NOR GATE : The NOR gate operates like an OR gate followed by an INVERTER.
